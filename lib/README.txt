Files needed in this "lib" folder:

1) qrcode.min.js   — download from:
   https://cdnjs.cloudflare.com/ajax/libs/qrcode-generator/1.0.3/qrcode.min.js
   Save it into this folder with the exact filename "qrcode.min.js".

That's it — only one file needed now. PDF export uses your browser's built-in
"Print > Save as PDF", which supports every language/script your OS has fonts
for (no font file to download, no jsPDF, fully offline).
